# Archive

This directory contains historical documentation and execution artifacts from earlier phases of the Wai & Watts project.

Nothing in this directory represents the current canonical architecture.
It is preserved intentionally as evidence of:

- Iterative architectural evolution
- AI-governed development process
- Convergence and remediation cycles
- Phase-based execution discipline

If you are reviewing this repository for architectural clarity, start with:

- `/docs` — Curated portfolio narrative
- `/engineering` — Canonical contracts, decisions, and governance artifacts

---

## Structure

### docs_legacy/

Pre-convergence documentation that previously lived under `/docs`.

Includes:
- Early architecture notes
- AI development onboarding artifacts
- Operational notes
- Reference materials
- Validation documentation

These materials reflect earlier states of the system and are retained for historical transparency.

---

### phase15_doc_convergence/

Artifacts from the Phase 15 architectural remediation and documentation convergence effort.

Includes:
- Architectural review findings
- Refactoring checklists
- Builder prompts
- Remediation plans
- Second-round verification findings

This represents a formal hardening pass on system boundaries and invariants.

---

### ai_dev_doc_convergence/

AI governance artifacts specific to the documentation convergence cycle.

Includes:
- Pre-convergence decision logs
- Progress snapshots
- Governance evolution notes

---

### legacy_phase_artifacts/

Standalone phase artifacts that were preserved independently of the main documentation set.

Examples:
- Early fixture generation documentation
- Phase 8 / Phase 9 ingestion artifacts

---

### phase_notes/

Narrative documentation tied to specific feature phases.

These reflect incremental product slices and system expansion efforts over time.

---

## Why This Exists

This project intentionally preserves its architectural evolution.

Rather than deleting prior states, the repository separates:

- Current canonical architecture
- Historical execution artifacts

This mirrors how mature teams preserve architectural history while maintaining a clean active surface area.
