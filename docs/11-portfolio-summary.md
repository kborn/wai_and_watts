# Wai & Watts — Portfolio Summary

## Overview

Wai & Watts is a lineage-first environmental data platform designed to ingest public environmental and energy datasets, preserve explicit dataset lineage, and generate grounded natural-language explanations using a fact-pack architecture that prevents hallucination.

The project was intentionally built as a portfolio artifact to demonstrate production-grade backend engineering, ingestion pipeline design, and disciplined AI integration patterns.

---

## Key Architectural Capabilities

Deterministic ingestion pipeline  
Publisher spreadsheets are transformed into canonical contract CSV files before ingestion, ensuring reproducibility and protecting against publisher format drift.

Explicit dataset lineage model  
Each ingestion produces an immutable dataset_release record linked to its originating dataset_source, allowing complete reconstruction of system state and safe re-ingestion.

Contract-driven architecture  
All ingestion is based on normalized contracts rather than raw publisher artifacts, isolating ingestion logic from external variability.

Grounded LLM explanation architecture  
LLMs operate exclusively on structured Fact Packs generated by deterministic database queries, preventing hallucination and ensuring explanations remain verifiable.

Full-stack implementation  
Backend: Java, Spring Boot, Postgres  
Frontend: React, TypeScript  
Infrastructure: Docker, CI pipelines, CLI ingestion tooling
